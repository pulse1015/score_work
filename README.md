# score_work

## 環境
- OS
    - Mac OS 10.14.5 (Mojave)
- 使用ソフト
    - musescore 3.1.0
- **注意**
    - musescore 2 では楽譜データが開けないため、musescore 3.1以上を使用すること。
    - ダウンロード先
        - https://musescore.org/ja/download

## ブランチ構成

### master
- 公開ブランチ

### develop
- 作業用ブランチの集約先
    - 集約後は `feature-yyyymmdd` ブランチを **削除** する

### feature-yyyymmdd
- 作業用ブランチ
